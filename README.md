# Dark Web: De 0 a Experto

**Curso universitario de análisis de la economía underground**  
*por Diego Saavedra*

## Descripción

Este curso explora el ecosistema de la dark web desde una perspectiva técnica de ciberseguridad. Los estudiantes analizan herramientas maliciosas, entienden el modelo económico del underground, y desarrollan habilidades de defensa basadas en conocimiento de amenazas.
