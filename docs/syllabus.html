3.2 2. Descripción

Este curso explora el ecosistema de la dark web desde una perseto técnica de investigación en ciberseguridad. Los estudiantes analizarán herramientas maliciosas, entenderán el modelo económico de la cibercriminalidad y desarrollarán habilidades para defender organizaciones contra amenazas emergentes. El enfoque combina teoría, análisis técnico y laboratorios controlados.

3.4 4. Metodología

El curso sigue el enfoque ABACOM de aprendizaje activo con: - 60% práctica: Laboratorios con ejecución manual de comandos - 30% teoría: Conceptos, casos de estudio y análisis - 10% autónomo: Lecturas, ejercicios y proyectos

Los laboratorios usan VMs completamente aisladadas. No se permite el copy-paste — cada comando se escribe manualmente para construir competencia real.
